# HACS Token Hub

自托管 GitHub Token 共享池，为 [HACS 极速版](https://github.com/hacs-china/integration) 提供共享令牌服务。

## 功能特点

- 🔑 自托管 Token 池，完全自主可控
- 🇨🇳 通过 `ghapi.hacs.vip` 代理验证，大陆网络可用
- 🌐 内置 Web 管理界面，方便添加和管理 Token
- 💾 数据持久化，重启不丢失
- 🔌 支持 HA Ingress，直接在侧边栏访问
