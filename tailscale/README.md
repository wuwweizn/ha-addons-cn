## 关于

Tailscale 是一种零配置 VPN，可在几分钟内安装在任何设备上，包括你的 Home Assistant 实例。

它可以在服务器、电脑和云实例之间创建一个安全网络。即使它们被防火墙或子网隔开，Tailscale 也能正常工作。Tailscale 会为你管理防火墙规则，无论你身在何处，都能安全访问网络。
