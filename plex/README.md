## 关于

Plex 插件将你最喜欢的媒体集中到一个地方，让你**美观且轻松地享受**它们。
本插件提供的 Plex 媒体服务器可以**整理你的个人视频、音乐和照片收藏**，并将其**流式传输到你的所有设备**。
