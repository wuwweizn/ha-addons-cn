## 关于

[Node-RED][nodered] 是一个用于将硬件设备、API 和在线服务以新颖方式连接在一起的编程工具。

它提供了一个**基于浏览器的编辑器**，可以轻松使用调色板中丰富的节点来构建流程，并可一键部署到运行时环境。
